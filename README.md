# Hi, I'm Kingsley 👋

Systems Engineer based in Chicagoland area. Currently engineering systems around the world, deep in Linux, Kubernetes, databases, and everything else the stack throws at me. Most of all, I’m driven by curiosity... always learning, always leveling up!

## 🛠 What I work on day to day

**Infrastructure & Virtualization**
- KVM hypervisor management - VM lifecycle, LVM storage, network bonding, VLAN config
- Multi-datacenter buildouts from the ground up (provisioning, networking, storage, DNS)
- SAN storage (iSCSI multipath) and load balancer configuration

**Kubernetes & Containers**
- RKE2/Rancher cluster management across global sites (40+ node clusters)
- EKS cluster upgrades and subnet/IP exhaustion remediation
- Helm chart development and rollouts

**Automation**
- Ansible role development for Linux and Windows server provisioning
- Jinja2 template development across multi-datacenter inventory structures
- CI pipeline execution for infrastructure automation

**Databases & Messaging**
- MongoDB replica set deployment and management (PSA topology, KVM and AWS EC2)
- Elasticsearch cluster recovery
- I enjoy writing a good SQL query!



**Monitoring**
- Icinga2 and Prometheus alerting. Working with check commands, exporters, alert tuning
- JVM heap and OOMKill investigations across Kubernetes workloads

**Windows Infrastructure**
- IIS and Windows application server deployment and troubleshooting
- Active Directory, SQL Server access, and domain-joined Linux (SSSD/PAM)
